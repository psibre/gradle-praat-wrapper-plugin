Gradle Praat Wrapper Plugin
===========================

[Unreleased]
------------

### Added
- Common base plugin, extended by native plugins
- Meta-plugin which dynamically applies OS-specific native plugin
- High-level `PraatExec` task type with flexible `script` parameter

### Changed
- Restored VCS URL in plugin metadata

[v0.2] - 2017-02-17
-------------------

### Added

- Documentation (README, LICENSE, CHANGELOG)
- Support for Windows (64-bit)
- provide Praat version as `praat.version` task property

[v0.1] - 2017-02-16
-------------------

### Initial release

Providing Praat v5.4.22 for
- macOs (64-bit)
- Linux (64-bit)
- Windows (32-bit)
